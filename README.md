# Aprendendo-SQLite
Nesse repositório ficaram os bancos de dados que vou fazer para aprender SQLite junto com python
no pycharm.

# Comentarios
As partes comentadas são comandos SQL que foram executados ou sao explicações dos comandos.

# Tecnologia utilizadas
**PyCharm** como IDE.  
**Python 3.13** com o sqlite3 nativo.  
**Git** para versionamento.  
**Git hub** para armazenamento em nuvem.  
**HTTPS** como protocolo de transferência.
